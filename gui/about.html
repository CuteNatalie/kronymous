About Krotor

Krotor v1.x.x
A Chrome app which creates a system wide (tor)socks proxy at port 9999 enabling access to Tor network.
Once the proxy is up and running(can be verified by logs in terminal) you can use any proxy extension in browser,system proxy settings etc to access tor socks proxy at 127.0.0.1:9999 (in next version this will be handled in app itself)

On windows, system proxy settings don't work, use a proxy extension such as foxyproxy

(the official codebase of tor is running in this app, this is port of Tor to NaCl platform on Chrome browser.)
