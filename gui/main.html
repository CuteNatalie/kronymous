Hi there!

Thanks for installing KroTor, with this app you can access internet via Tor. [visit any blocked websites, browse anonymously etc]
Once started, tor socks proxy will be available on your system which can be accessed at 127.0.0.1:9999
This app is open sourced, for details click here.

Instructions
In order to use Tor Socks proxy created by this app, please configure any extension/system settings/apps accordingly.
Following instructions are only applicable for Google Chrome.
Chrome Apps (such as this one) cannot set the proxy settings in the Chrome browser, only extensions can do so.As an example, one can use extension ProxySwitchyOmega(open sourced)
Setup This is one time process.
Install ProxySwitchyOmega or any proxy extension.
Create a proxy profile named KroTor with following settings.
protocol : socks5
server : 127.0.0.1
port : 9999 (or the tor port you defined in settings)
Using it
1. Run KroTor, click on Start Tor, make sure you see "Bootstrapped 100%" in terminal window(app's middle section with black background)
2. Enable KroTor proxy in SwitchyOmega's dropdown menu(click its icon i.e. black O on right side of address bar)

To verify that you are on Tor network, please visit : https://check.torproject.org

If Tor takes too long to connect, try enabling/modifying accessible ports in settings.

If you face any issue, please ask on the support page.

Advance options.
If you want set exitnodes, provide username/password for (network)proxy etc, you can do so by enabling advance options in settings and provide them in format OPTIONS values
As an example:
For exitnodes: ExitNodes node,node,â€¦
For username/password for (network)proxy: HTTPSProxyAuthenticator username:password (this is stored in cleartext in your browser locally.
You can give multiple options, make sure there is a space between them i.e. OPTIONS1 values1 OPTIONS2 values2 ...
For more details see Tor manual

This app lacks the functionality to configure custom bridges, pluggable transports. There is plan to implement it but I can't give ETA.

Please note that this Chrome App is not a substitute of Tor Browser Bundle

Instructions to check if you are accessing internet via Tor and browsing anonymously.
Before proceeding make sure that you have configured everything correctly, click help button to see the instructions.

To verify that you are on Tor network, please visit : https://check.torproject.org

To check if your local ip (your unique identifier on internet) is leaking or not please visit : https://ipleak.net
To avoid webrtc leaks follow instructions given at https://ipleak.net/#webrtcleak
Note to window users:
Window's system proxy settings are unreliable and on top of that DNS queries are are also unreliable on it.

As a general rule:
Make sure you use KroTor only with those applications/operating systems which have support for socks4/socks5 proxy support (remote dns resolution). Example: for Google Chrome you can use switchyomega extension.

Remember anonymity and privacy are two different things.

Note from developer ( Deepankar Tyagi ):
These are just my opinions and are not endorsed by anyone.
Two main factors are responsible for being anonymous :
1. The traffic coming to and from the user's system must not be traceable back to user's system.(The way in which the user's system connects to outside world must not lead to disclosure of user's identity)
2. The action of user must not lead to disclosure of user's identity.
KroTor(Tor Network) takes care of first part and user is responsible for second part.
IMHO, no one can guarantee a 100% anonymity or for that matter I definitely cannot(because of second part), that being said it is highly likely that user will remain anonymous without any extra steps other than setting KroTor correctly.
To increase the chances of being anonymous on chrome, run chrome in incognito mode with all the extensions(except proxy extension) disabled, password auto fetching disabled and avoid using applications/websites which do not honor proxy/privacy settings.(you can disable the JavaScript also) For more details visit: https://www.torproject.org/download/download#warning
